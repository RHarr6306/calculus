X Approaches:
lim_[x -> c](...)
Function of X:
lim_[x -> c](f(x))

Note

Use * for multiplication. For example, use 3*x instead of 3x or 3(x). Odd results may occur otherwise. Be sure to use parenthesis in trigonometric functions as well.

Evaluating Limits

  1. f(x) is defined at c
  2. The limit of f(x) at c exists
  3. f(c) = lim_[x->c](f(c))
